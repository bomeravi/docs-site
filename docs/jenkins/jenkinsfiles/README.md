


Jenkinsfile templates for common application types.

Copy the relevant file to your repository root as `Jenkinsfile` and update variables (registry, credentials IDs, build/test commands).

Files:
- `Jenkinsfile-php` — PHP (Composer, PHPUnit)
- `Jenkinsfile-laravel` — Laravel (Composer, artisan)
- `Jenkinsfile-wordpress` — WordPress (container build)
- `Jenkinsfile-react` — React (npm build, static hosting)
- `Jenkinsfile-java` — Java microservice (Maven/Gradle)
- `Jenkinsfile-go` — Go (build/static binary)
- `Jenkinsfile-python` — Python (pytest)
- `Jenkinsfile-django` — Django (migrations, collectstatic, gunicorn)
